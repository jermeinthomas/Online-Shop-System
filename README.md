# Online Shop System

A lightweight Python module for handling discount logic in an online store.  
It evaluates customer type and purchase amount to determine the correct discount.

## Features
- Supports **VIP**, **Regular**, and **Employee** customers  
- Automatic discount calculation  
- Clean, modular structure using a dedicated `get_discount()` function  
- Easy to reuse or integrate into larger systems

## How It Works
- Map customer types to predefined discount rates  
- Validate amount thresholds  
- Return the correct discount value for further processing

## Usage
Run the script:

```bash
python Online-Shop-System.py
